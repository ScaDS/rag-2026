# Example workflows

The workflows are intended to show the process of using the HPC system and
 not to provide a 'state of the art analysis' examples.
 Please note that used modules, parameters are highly dependent on your research question,
 species and input data. Therefore, this analysis might not be appropriate for your project.
 We do not guarantee for correctness.
